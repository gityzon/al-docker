## 使用免费 MySQL 远程数据库（推荐）

由于 Herok 免费账户无法永久保存应用数据，在没有使用正确的方法防止应用休眠情况下，Alist 会被重置。使用远程 MySQL 则可以解决这个问题。

注册免费 MySQL 远程数据库：https://www.db4free.net/

完成注册后，你的邮箱会收到MySQL 连接地址、端口、数据库名称、用户信息。
